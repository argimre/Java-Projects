# Java-Projects
Welcome to my Java Projects repository! This is a collection of various Java applications that I've created to explore different concepts and enhance my skills. The projects range from simple games to more complex tools, showcasing various programming techniques and problem-solving approaches that I've learned along the way.

Feel free to explore the individual projects and use them as learning resources or inspiration.

## Projects
Here are the Java projects available in this repository:

- `Library Management`: A system to manage books and users in a library, with features for adding, removing, and searching books.
- `Recursive Calculator`: A calculator application that uses recursion to solve mathematical expressions.
- `Text Glossary`: A program that creates a glossary website with the given terms and definitions.
- `Tic Tac Toe`: A classic 3x3 Tic-Tac-Toe game with a simple command-line interface.
- `Web Scraper`: A tool to scrape data from web pages and put them on an HTML page.
- `Word Counter`: A utility to count the frequency of words in a given text.
- `Word Guesser`: A word-guessing game where the player must guess a hidden word.

## Requirements
- JDK 8 or higher
- An IDE like Eclipse or VS Code
- Maven or Gradle (if necessary for project dependencies)
